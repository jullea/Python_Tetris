# Python_Tetris
The purpose of this project is to create Tetris game code with OOP(Object Oriented Programming).

## Requirements
Code is written in Python 3.6 and requires:
* _Pygame 1.9.6+_
* _Numpy 1.18.1+_

## Installation
**PyGame**
```
pip install pygame
```
**Numpy**
```
pip install numpy
```
